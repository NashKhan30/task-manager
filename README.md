# 📋 Task Manager

A responsive Task Manager web application built using **HTML, CSS, and Vanilla JavaScript**.

This project was built to strengthen my understanding of JavaScript fundamentals, DOM manipulation, CRUD operations, Local Storage, responsive design, and Git/GitHub workflow.

---

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Dashboard with:
  - Total Tasks
  - Pending Tasks
  - Completed Tasks
- Task category selection
- Dark / Light mode
- Data persistence using Local Storage
- Fully responsive design
- Instant dashboard updates after every action
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

- Vanilla JavaScript (ES6)

- GitHub
